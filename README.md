# Quick Blur (Clipboard In/Out, PWA)

素早く画像に「ぼかし／モザイク」を加えられる **完全クライアントサイド** の Web アプリです。  
クリップボードの画像を貼り付けて編集し、再びクリップボードへコピー（またはPNG保存）できます。  
PWA 対応（インストール／オフライン起動）。

## 特長
- 📋 **クリップボード入出力**：貼り付け（⌘/Ctrl+V）、書き出し（⌘/Ctrl+C）  
- 🖌️ **ブラシ操作**：ぼかし／モザイク／消しゴム、サイズと強さ調整  
- ↩️ **Undo 対応**：⌘/Ctrl+Z  
- 💾 **PNG 保存** ボタン  
- 🔐 **完全ローカル処理**：画像はネットへ送信されません  
- 📱 **PWA 対応**：インストールしてオフライン動作

## 使い方
1. ブラウザで `index.html` を開く（GitHub Pages からの利用も可）  
2. **画像を貼り付け**（⌘/Ctrl+V）または「画像を開く」or D&D  
3. キャンバスをドラッグして顔などを塗る  
4. 仕上がりを **クリップボードへコピー** または **PNG 保存**

### 主なショートカット
- 画像貼り付け：⌘/Ctrl+V、または `P`  
- クリップボードへ出力：⌘/Ctrl+C、または `C`  
- Undo：⌘/Ctrl+Z  
- ブラシサイズ変更：`[` / `]`  
- ダウンロード：`D`

## ブラウザ対応・注意
- 近年の Chrome / Edge / Firefox で動作。  
- **Safari / iOS** では、画像のクリップボード **書き出し** が制限される場合があります。  
  その際は PNG 保存をご利用ください。  
- クリップボード入出力にはブラウザ側の**許可**と**ユーザー操作**が必要な場合があります。

## PWA（インストール方法）
1. HTTPS で `index.html` にアクセス  
2. ブラウザの **「インストール」** もしくは **「ホーム画面に追加」** を実行  
3. 次回からホーム画面/Dock から **全画面アプリ風** に起動できます  
   - オフラインでも起動可（初回読込後）

## ローカル開発
```bash
python3 -m http.server 8000
# http://localhost:8000
```

## GitHub Pages で公開
1. このリポジトリを GitHub に push  
2. Settings → Pages → Branch を `main` / `/ (root)` に設定  
3. 公開URL: `https://<ユーザー名>.github.io/<リポジトリ名>/`  
   - サブパス配信なら `manifest.json` の `start_url` を `"/<リポジトリ名>/"` に調整
